# **App Name**: CognitoLearn

## Core Features:

- Secure Authentication & Role Management: Users can sign up, log in securely with Firebase Authentication, and select/be assigned Student or Teacher roles, securing access to appropriate dashboards and features.
- Interactive Role-Based Dashboards: Provide distinct, modern dashboards for Students (to view content, take quizzes) and Teachers (to manage classes, upload content) ensuring a tailored user experience.
- AI-Powered Adaptive Learning Tools for Students: Generative AI tools provide personalized study aids, including dynamic note summaries, tailored video recommendations, and practice quizzes to reinforce understanding.
- AI-Driven Insights & Management for Teachers: Utilize AI to analyze student performance data, offering tool-based suggestions for teaching strategies and automatically organizing students into effective study groups.
- Classroom & Content Management: Teachers can efficiently create and manage classes, upload educational videos and notes, and design quizzes, while students can join classes and access learning materials.
- Performance Analytics & Progress Tracking: Dedicated analytics dashboards for both students and teachers to visualize individual and class performance trends over time, including marks, improvement metrics, and weak areas.
- Direct Teacher-Student Communication: A real-time messaging interface allows students to directly chat with their teachers, fostering collaboration and timely assistance.

## Style Guidelines:

- Primary color: A calming and professional muted blue (#337EAA) symbolizing intelligence and trust in an educational context.
- Background color: A very light, desaturated blue-grey (#F0F6F9) providing a clean and minimal backdrop that promotes focus and readability.
- Accent color: An energetic and attention-grabbing violet (#4D4DD9) for interactive elements and highlights, offering strong contrast against the primary and background.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for extensive text and data displays in a professional EdTech setting.
- Utilize a consistent set of minimalist line icons for navigation and functional elements, maintaining a professional and uncluttered aesthetic aligned with the EdTech theme.
- Implement a clear sidebar navigation system for easy access to different sections of the dashboard. Employ a modular, grid-based layout for content areas, ensuring efficient organization and visual hierarchy.
- Incorporate subtle hover effects on interactive elements and smooth transitions between page states to enhance the user experience without being distracting. Implement clean loading animations for data-intensive sections.
